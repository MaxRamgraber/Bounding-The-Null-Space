# Bounding-The-Null-Space
This repository contains the code required to reproduce the examples in the manuscript "Bounding the Null Space: Interval-Based Uncertainty Quantification for Non-Identifiable Groundwater Models".

The folder "figures" includes the manuscript's figure files, as well as the Python code used to generate them. Likewise, the folder "experiments" includes three subfolders with the required Python code to reproduce the simulations in the manuscript's experimental section.
